# Functional and Logic Programming (PFL)

**Course**: Functional and Logic Programming (L.EIC024)  
**Institution**: FEUP - Faculdade de Engenharia da Universidade do Porto  
**Academic Year**: 2025/2026  
**Professor**: Pedro Vasconcelos, DCC/FCUP

## Projects

This repository contains the practical assignments developed for the Functional and Logic Programming course:

- **Project 1**: Extended Calculator in Haskell - Implementation of a parser-based calculator with arithmetic operations and variable assignments.
- **Project 2**: Uluru Puzzle Solver in Prolog - Logic programming solution for constraint satisfaction puzzles using backtracking.



## Technologies

- **Haskell**: Functional programming with parser combinators
- **Prolog**: Logic programming with constraint satisfaction
- **SWI-Prolog**: Prolog implementation with `library(lists)`
